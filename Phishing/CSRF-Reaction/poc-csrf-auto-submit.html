[CSRF] Zero-Trust Browser Reaction Test

Tests browser reaction to Cross-Site Request Forgery attacks — forged requests sent from an attacker-controlled page to a target site.

Target Configuration

Test 1: Auto-Submitting POST Form

Creates a hidden form that auto-submits a POST request to the target with the victim's cookies.

Test 2: GET-Based State Change

Uses IMG, SCRIPT, and LINK tags to fire GET requests with the victim's cookies.

Test 3: XMLHttpRequest / Fetch CSRF

Tests cross-origin requests with credentials (cookies). Browser should enforce CORS.

Test 4: Clickjacking-Assisted CSRF

Embeds target in an iframe with transparent overlay — victim clicks thinking they interact with attacker page.

Test 5: SameSite Cookie Bypass Techniques

Tests various techniques to bypass SameSite=Lax cookie restrictions.

Test 6: Content-Type Variations

Tests CSRF with different content types that don't trigger CORS preflight.
